# 🚀 Quick Reference Card

## 3 Steps to Go Live

### Step 1️⃣ Add Your Image (2 minutes)
- Save photo as `profile.jpg` in portfolio folder
- Should be square (500x500px)
- Refresh browser - you'll see it in the circle!

### Step 2️⃣ Create GitHub Repository (5 minutes)
1. Go to [github.com/new](https://github.com/new)
2. Name: `your-username.github.io` (EXACT match!)
3. Set to **Public**
4. Create

### Step 3️⃣ Upload Files to GitHub (2 minutes)
1. Click **"Add file"** → **"Upload files"**
2. Drag and drop: `index.html`, `profile.jpg`
3. Click **"Commit changes"**
4. Wait 2-5 minutes...

✅ **DONE!** Visit: `https://your-username.github.io`

## ❓ Common Issues

| Issue | Solution |
|-------|----------|
| Image not showing | Name file exactly `profile.jpg` |
| Portfolio not live | Wait 5+ mins, check repo name is `username.github.io` |
| Page looks blank | Hard refresh: Ctrl+Shift+R (Windows) or Cmd+Shift+R (Mac) |
| Contact form broken | Set up Formspree.io account, update form ID |
